openSUSE

openSUSE — это свободная операционная система на базе Linux для Вашего ПК, ноутбука или сервера. Вы можете просматривать веб, управлять почтой и фотографиями, выполнять офисную работу, смотреть видео или слушать музыку. Have a lot of fun!

  •   Get it: Загрузите openSUSE или дополнительное ПО.  
  •   Discover it: Узнайте больше об openSUSE с помощью документации и других учебных материалов!  
  •   Create it: Создавайте пакеты для всех ключевых дистрибутивов Linux с помощью Build Service.
