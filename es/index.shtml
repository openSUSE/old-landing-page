openSUSE

openSUSE es un sistema operativo libre basado en Linux para PC, Portátil o Servidor. Puedes navegar por internet, administrar correo electrónico, fotografías, trabajos de oficina, reproducir música y videos, y mucho más.

  •   Consíguelo: Descarga openSUSE o encuentra software adicional.  
  •   Descúbrelo: Aprende más sobre openSUSE, con tutoriales, documentación, y más!  
  •   Constrúyelo: Crea paquetes para las distribuciones más importantes de Linux utilizando el Build Service.
