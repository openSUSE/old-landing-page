openSUSE

To openSUSE είναι ένα ελεύθερο και βασισμένο στο Linux λειτουργικό σύστημα για τον σταθερό ή τον φορητό σας υπολογιστή ή για τον server σας. Μπορείτε να σερφάρετε στο διαδίκτυο, να διαχειριστείτε τα e-mails και τις φωτογραφίες σας, να κάνετε δουλειά γραφείου, να αναπαράγετε βίντεο ή μουσική και να το διασκεδάσετε!

  •   Αποκτήστε: Αποκτήστε την openSUSE ή βρείτε επιπλέον λογισμικό.  
  •   Ανακαλύψτε: Μάθετε περισσότερα για την openSUSE με μαθήματα, τεκμηρίωση και πολλά ακόμη!  
  •   Δημιουργήστε: Δημιουργήστε πακέτα για όλες τις κύριες διανομές Linux μέσω του Build Service.
