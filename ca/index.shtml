openSUSE

OpenSUSE és un sistema operatiu lliure basat en Linux per a ordinadors, portàtils o servidors. Podeu navegar per Internet, gestionar el correu electrònic i les fotografies, fer feina d'oficina, mirar vídeos o escoltar música i passar-vos-ho molt bé!

  •   Obtingueu-lo: Descarregeu l'openSUSE o trobeu-ne programari addicional.  
  •   Descobriu-lo: Apreneu més sobre l'openSUSE amb tutorials, documentació i molt més!  
  •   Creeu-lo: Construïu paquets per a les grans distribucions Linux amb el Build Service.
